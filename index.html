Desafios do Módulo 2: Fundamentos do CSS

Desafio #1: Mudar a cor do texto

Use CSS para mudar a cor do texto abaixo para vermelho:

Você pode usar "cor: vermelho;" (português) ou "color: red;" (inglês).

Este texto deve ficar vermelho quando você aplicar o estilo correto.

No mundo profissional, usamos termos em inglês: "color" para cor e "red" para vermelho.
Solução do Desafio 1:
#text-to-style {
    color: red; /* Em CSS usamos termos em inglês: "color" para cor e "red" para vermelho */
}
Editor de CSS

Desafio #2: Mudar a cor de fundo do site

Use CSS para mudar a cor de fundo do site para azul claro:

Você pode usar "fundo: azulclaro;" (português) or "background-color: lightblue;" (inglês).

O fundo do site inteiro deve mudar de cor quando você aplicar o estilo correto.

No mundo profissional, usamos termos em inglês: "background-color" para cor de fundo e "lightblue" para azul claro.
Solução do Desafio 2:
body {
    background-color: lightblue; /* Em CSS usamos termos em inglês: "background-color" para cor de fundo */
}
Editor de CSS

Desafio #1: Centralizar um elemento

Use CSS para centralizar a caixa abaixo:

Dica: Use margens automáticas (margin: auto) e defina uma largura.

Esta caixa deve ser centralizada horizontalmente.

Solução do Desafio 1:

    margin: 0 auto;
    width: 50%;
}
